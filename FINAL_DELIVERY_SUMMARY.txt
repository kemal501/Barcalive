📞 TROUBLESHOOTING QUICK REFERENCE:

"Cannot find module express" → Run: npm install
"Firebase credential error" → Check .env values
"Cannot connect to localhost:5000" → Backend not running
"404 Not Found" → Wrong API_BASE URL
"CORS error" → Add URL to CORS_ORIGIN in .env
"Room not found" → Create new room, refresh
"Coins not updating" → Check Firebase rules
